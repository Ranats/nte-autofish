# Support

For setup or compatibility issues, open a GitHub issue in this repository.

Please include:

- Windows version
- Python version
- Screen resolution and display scale
- NTE display mode, preferably borderless fullscreen
- Whether `dry_run.bat` detects the expected fishing state
- A short description of the screen where the automation stops

Do not post private account information, purchase receipts, license emails, or UID screenshots publicly.

## Refunds

Payments and delivery may be handled through itch.io or LemonSqueezy depending on where you purchased. Use the purchase receipt or the store's support flow for payment-related requests.

## 日本語

セットアップや互換性の問題は、このリポジトリの GitHub Issue に投稿してください。

以下を添えると確認しやすくなります。

- Windows バージョン
- Python バージョン
- 解像度と表示スケール
- NTE の表示モード
- `dry_run.bat` が釣り待機状態を検出しているか
- どの画面で自動化が止まるか

公開Issueには、個人情報、購入レシート、ライセンスメール、UID が映ったスクリーンショットを投稿しないでください。

決済、配布、返金に関する問い合わせは、購入したストアのレシートまたはサポート導線を使用してください。
